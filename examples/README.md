# Examples Directory Guide

Examples are organized by language and by the same hierarchy used in `agd_docs`.

## Structure

- `ko/10_source/*`
- `ko/20_derived/*`
- `ko/30_shared/*`
- `en/10_source/*`
- `en/20_derived/*`
- `en/30_shared/*`

## Folder Meaning

- `10_source`: source-of-truth examples
- `20_derived`: examples derived from source docs
- `30_shared`: shared planning/coordination examples

Current path conventions (sync with CLI):

- `postmortem` docs: `30_shared/postmortem/`
- `maintenance-case` docs: `30_shared/maintenance/`
- `incident-case` docs: `30_shared/errFix/`

## Index Files

- Korean showcase index: `TEMPLATE_SHOWCASE_INDEX_ko.md`
- English showcase index: `TEMPLATE_SHOWCASE_INDEX_en.md`
